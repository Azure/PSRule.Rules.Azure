# v1.40

## New home for samples

Previously PSRule samples for Azure where hosted in a separate repository.
This added an unnecessary level of separation that made them harder to discover and added additional maintenance.

In the release we moved the samples into the PSRule for Azure repository and added an index.

We hope this will be the first of a series of changes that will make it easier for the community to contribute.
