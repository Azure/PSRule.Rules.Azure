# Use allowed regions

## SYNOPSIS

Resources should be deployed to allowed regions.

## DESCRIPTION

Resources should be deployed to allowed regions.

## RECOMMENDATION

Resources should be deployed to allowed regions.

## NOTES

By default, all regions are allowed.
To configure this rule:

- Override the `Azure_AllowedRegions` configuration value to set the allowed regions for validation.
