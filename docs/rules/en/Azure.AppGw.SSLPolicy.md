# Application Gateways use a minimum TLS 1.2

## SYNOPSIS

Application Gateway should only accept a minimum of TLS 1.2.

## DESCRIPTION

Application Gateway should only accept a minimum of TLS 1.2.

## RECOMMENDATION

Consider configuring Application Gateway to accept a minimum of TLS 1.2.
